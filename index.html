🐉 Quiz-redaktør

Tilføj, ret og slet quizzer i den centrale database — uden kode.

Forbindelse: ingen nøgle
🔑 Sådan laver du din engangs-nøgle (kun én gang) 1) Åbn github.com/settings/personal-access-tokens/new
2) Repository access → "Only select repositories" → vælg dragelaeseren-quiz
3) Permissions → Repository permissions → Contents → Read and write
4) Tryk "Generate token", kopiér den lange tekst, og indsæt den herunder.
